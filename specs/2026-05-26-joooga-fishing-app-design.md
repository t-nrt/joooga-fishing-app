# 「Joooga」フカセ釣りポイント推薦アプリ設計書

## 概要

「Joooga」は城ヶ島を拠点としたフカセ釣りのポイントアドバイスWEBアプリです。気象・海況・釣果データを総合的に分析し、最適な釣り場を3つ推薦します。

## 要件一覧

### 基本要件
- **アプリ名:** Joooga
- **UI:** スマホ最適化（PWA対応）
- **対象釣法:** フカセ釣り
- **対象魚種:** 選択式（グレ専用・磯釣り全般・回遊系・チヌ狙い）
- **主要機能:** 3つのポイント推薦、詳細情報表示、オフライン対応

### エリア対応
- **選択方式:** プルダウン式
- **デフォルト:** 城ヶ島
- **対応エリア:** 城ヶ島 → 三浦半島全体 → 相模湾エリア → 千葉エリア
- **動的処理:** エリア選択時にリアルタイムリサーチ実行

### データソース
#### 釣果情報
- 爆釣会議、釣果速報、アングラーズ
- 個人ブログ（RSS/シンジケート）
- X（Twitter）API（ハッシュタグ・位置情報検索）

#### 気象・海況データ
- 気象庁API（気温、風向・風速）
- kansoku-data.net（海水温）
- 潮汐API（潮回り、満干時刻）
- 波浪API（波高、波向）

#### タナ・地形情報
- 釣りドコ
- 海底図API
- ポイント詳細情報（動的取得）

### 更新頻度
- **リアルタイム更新:** 15分〜1時間間隔
- **データ処理:** 収集 → 正規化 → スコア算出 → ポイント推薦

### 法的・倫理的コンプライアンス

- **フロントエンド:** Next.js + PWA

4. 結果統合・表示

## ポイント推薦アルゴリズム

### スコア算出方式
**総合スコア = 気象スコア × 0.3 + 海況スコア × 0.3 + 釣果スコア × 0.4**

#### 気象スコア（30%）
```
- 風速: 3m/s以下 → +20点、5m/s以下 → +10点、以上 → -10点
- 風向: ポイント基点評価
  • オフショア → +15点
  • サイド → +5点  
  • オンショア → -5点
- 気温: 適温範囲 → +10点、極端 → -5点
```

#### 海況スコア（30%）
```
- 波高: 1m以下 → +20点、1.5m以下 → +10点、以上 → -10点
- 海水温: 季節適温 → +15点、前週比上昇 → +5点
- 潮回り: 中潮・大潮 → +10点、小潮 → +5点、長潮 → +0点
```

#### 釣果スコア（40%）
```
- 直近3日の釣果 → +25点（数・サイズ・魚種で重み付け）
- 1週間の釣果傾向 → +15点
- 同条件下での過去実績 → +10点
```

### 魚種別重み付け
```
- グレ: ×1.0 (基準)
- チヌ: ×0.9 (高価値)
- イサキ: ×0.8 (群れで期待大)
- アジ・サバ: ×0.6 (数釣り)
- その他: ×0.7
```

### 風向き評価ロジック
```javascript
function evaluateWind(windDirection, pointFacing) {
  const angleDiff = calculateAngleDifference(windDirection, pointFacing);
  
  if (angleDiff <= 45) return 15;  // オフショア
  if (angleDiff <= 135) return 5;  // サイド
  return -5;  // オンショア
}
```
